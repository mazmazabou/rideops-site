# CLAUDE.md — rideops-site

Marketing website for RideOps at https://ride-ops.com

## Tech Stack
- **Framework:** Next.js 16 (App Router) + React 19 + TypeScript
- **Styling:** Tailwind CSS 4 (via `@tailwindcss/postcss`)
- **Fonts:** DM Serif Display (headings) + DM Sans (body) via next/font
- **Icons:** Lucide React (installed, minimal usage)
- **Analytics:** Vercel Analytics + Speed Insights
- **Deployment:** Vercel (auto-deploys on push to `main`)

## Project Structure
```
app/
  page.tsx        — Homepage (single page site)
  layout.tsx      — Root layout (Navbar, Footer, Analytics)
  globals.css     — Tailwind theme + custom colors
  icon.svg        — SVG favicon (steel blue circle, white "RO")
components/
  Hero.tsx        — Hero section with dispatch screenshot
  Navbar.tsx      — Sticky nav with mobile menu ("use client")
  SocialProof.tsx — Campus social proof strip
  RiderCarousel.tsx — 3-frame booking wizard carousel ("use client")
  FeatureSection.tsx — Reusable feature block (text + screenshot)
  BrowserFrame.tsx — Desktop browser mockup wrapper
  PhoneFrame.tsx  — iPhone mockup wrapper
  ThemeShowcase.tsx — 4-campus theme grid
  FinalCTA.tsx    — Bottom CTA section
  Footer.tsx      — Footer links
public/screenshots/ — Marketing product screenshots
public/demo/        — Detailed demo screenshots (not used on homepage)
```

7. **Theming showcase** — "Your campus, your brand" (distinct #F0F5FA background)
8. Rides management — "Every ride, every status, every driver" (bg-gray-50)
9. Final CTA — "Ready to modernize your campus transportation?"
10. Footer

## Key Design Decisions
- **Rider section uses 3-frame carousel** (`RiderCarousel.tsx`) with `rider-step1-where.png`, `rider-step2-when.png`, `rider-step3-confirm.png`
- **Desktop:** 3 phones side-by-side with step labels underneath
- **Mobile:** CSS scroll-snap carousel with dot indicators, auto-advance every 4s, pauses on touch
- **Dispatch feature section removed** — hero screenshot already covers dispatch, avoids duplication
- **Alternating section backgrounds** for visual rhythm (white → gray-50 → white → gray-50)
- **Theming section** has distinct blue-tinted background (#F0F5FA) and extra vertical padding
- **Favicon:** SVG icon (steel blue #4682B4 circle with white "RO" text), auto-detected by Next.js at `app/icon.svg`
